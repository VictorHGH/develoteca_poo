# Develoteca POO

Repositorio de ejercicios practicos de Programacion Orientada a Objetos en PHP.

## Contenido

- Archivos `001_*.php` a `008_*.php` con ejercicios y retos.
- `index.php` como punto de entrada rapido.

## Requisitos

- PHP 8.x
- Servidor local (por ejemplo el stack Docker de `~/Documents/programacion/php`)

## Ejecucion

Si usas el entorno Docker padre (`php`), abre:

- `http://127.0.0.1:8082/develoteca_poo/`

O ejecuta scripts individuales desde CLI:

```bash
php 001_ejercicio1.php
```

## Objetivo

Consolidar conceptos base de POO en PHP mediante archivos independientes y faciles de revisar.
